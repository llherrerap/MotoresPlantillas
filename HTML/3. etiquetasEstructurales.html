Etiquetas Semanticas Estructurales

Son las etiquetas que permiten organizar el codigo para que el navegador pueda identificar las secciones

Div es solo una etiqueta contenedora que no tiene ninguna semantica
header: Cabecera del sitio web o de una seccion
main: Define la seccion principal del documento, solo puede existir una etiqueta main por documento
footer: define el pie de pagina del sitio web o de una seccion
nav: representa la navegacion/botonera del sitio web
article: representa una sección autocontenido (que por si sola se explcia)
aside: representa contenido complementario o secundario
section: representa una seccion del codigo generico
address: representa una informacion de contacto

Etiquetas de Bloque vs Etiquetas de Línea

La etiqueta de bloque por excelencia es div. No tiene ningun peso semantico, una etiqueta de bloque es aquella que genera saltos de linea con sus etiquetas hermanas. Ocupa todo el ancho disponible del navegador
La etiqueta de linea por excelencia es span. No tiene ningun peso semantico. Una etiqueta de linea solo ocupa el espacio necesario que tiene su contenido. No genera salto de linea ni arriba ni abajo Hola! Soy otra etiqueta span
